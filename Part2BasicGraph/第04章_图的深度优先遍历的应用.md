# 第04章 图的深度优先遍历的应用

> 常见的应用如下

+ 求图的连通分量(4.1~4.2)
+ 求两点间是否可达(4.3)
+ 求两点间的一条路径
+ 检测图是否有环
+ 二分图检测
+ 寻找图中的桥和割点
+ 哈密尔顿路径
+ 拓扑排序

> DFS递归没退出一次，说明图有了一个连通分量，所以在dfs()下方，连通分量个数connectedComponentCount加1即可
